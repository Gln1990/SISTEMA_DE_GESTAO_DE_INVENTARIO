# 💻 Sistema de Gestão de Inventário - Hardware

Este projeto é um sistema de gerenciamento de notebooks desenvolvido em **Linguagem C**. Ele foi projetado para simular o controle de estoque e manutenção de equipamentos, unindo conceitos de programação estruturada com a experiência prática em suporte técnico de hardware.

## 🚀 Funcionalidades

- **Cadastro de Equipamentos**: Registro de modelo e número de série com geração automática de ID.
- **Relatório de Inventário**: Listagem formatada de todos os itens cadastrados.
- **Busca Inteligente**: Localização de equipamentos por Número de Série (S/N).
- **Controle de Status**: Alteração dinâmica entre *Disponível*, *Manutenção* e *Alocado*.

## 🧠 Conceitos Aplicados

Durante o desenvolvimento deste projeto, apliquei conceitos fundamentais de Análise e Desenvolvimento de Sistemas:

1. **Structs**: Agrupamento de dados heterogêneos para representar um objeto do mundo real (Notebook).
2. **Passagem por Referência**: Uso de ponteiros para manipulação eficiente de contadores globais.
3. **Modularização**: Organização do código em funções específicas para facilitar a manutenção.
4. **Manipulação de Strings**: Uso da biblioteca `string.h` para comparação e armazenamento de dados textuais.
